## Jackramp / jackUSD

### Documentation
For more detailed documentation, please refer to the following [Gitbook link](https://kbaji.gitbook.io/jackramp).

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
```

### Deploy

```shell
$ forge script DeployJackrampScript --rpc-url <your_rpc_url> --verify --broadcast
```
